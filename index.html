The Public Suffix List - Domain name suffixes for browsers

The Public Suffix List is a cross-browser initiative to provide an accurate list of domain name suffixes for internet browser software.

Amongst other uses, the Public Suffix List will help browsers with:

  • Avoiding privacy-damaging "supercookies" being set for high-level domain name suffixes (such as .co.uk)
  • Highlighting key parts of a domain name in the user interface
  • Accurately sorting history entries by site

Since maintaining a list of all domain name suffixes would be a colossal task, we would like domain registries to help out by maintaining their own section of the List. Registries can learn more about the List and submit amendments using the correct format for submissions. Browser manufacturers that are interested in using the List in the product can learn how to.
