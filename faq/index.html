Frequently Asked Questions

How is the list produced?
By the hard work of Mozilla project volunteers, and by submissions from registries - to whom we are very grateful.
How can I update or alter the list?
The Submit amendments page outlines the process for requesting changes to the list.
What software uses the list?
The following software is known to use the list:
  • Google Chrome 1.0 and later
  • Mozilla Firefox 3.0 and later
  • Opera 10 and later
  • reg-dom libs - C, Perl and PHP free software libraries for adding PSL support to your application
